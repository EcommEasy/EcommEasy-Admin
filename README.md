# Ecomm-Admin - A React and NodeJS based Ecommerce Dashboard Application

Ecomm-Admin is React and Node.js based eCommerce platform. 

Built with:
* Node.js v8.9
* React v16
* Redux
* Express
* Babel
* WebPack 4
* MongoDB

## Dashboard
Client-side dashboard use JSON Web Token (JWT) to access REST API.

## Installation

### Requirements
* Node.js >= 8
* MongoDB >= 3.2
